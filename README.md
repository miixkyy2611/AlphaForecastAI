# AlphaForecast AI

Forecasting bursátil con Python 3.13, Flask y ensemble de modelos tree-based. Esta edición incluye una capa visual más premium con dashboard glassmorphism, temas de color, microanimaciones, plots oscuros integrados con la paleta y una vista de resultados más limpia para demos y portfolio.

## Qué trae
- Forecast multi-step directo sin TensorFlow
- Backtest temporal real
- Bandas de incertidumbre 80% y 95%
- Feature importance y pesos del ensemble
- UI más estilizada con selector de tema local
- Gráficas oscuras alineadas al dashboard

## Correr localmente
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## Nota
Es una herramienta experimental de forecasting cuantitativo. No constituye recomendación de inversión.
